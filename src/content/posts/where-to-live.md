# 麻布十番
- 今住んでいる場所
- 住み心地は良く満足している
    * どこにでも行きやすい
        + フリーランスエンジニアで家で作業をする人でも気分転換にカフェとかに行きやすい
        + 出かける時もフットワークが軽くなる
    * 人が多すぎず少なすぎない

# 六本木
- 麻布十番の横。
- もう少し利便性を上げたい場合はこちらが良さそう。
    * 六本木にもIT系の会社がある
    * 渋谷にも道路をまっすぐ行くだけだからアクセスが良い。バスもある
* 住みやすそうかは微妙な感じ

# 虎ノ門
* 作業カフェ目当て。
* 平日は人が多いが休日は静か
* 新橋も徒歩圏内なのでアクセスも良い。
* 麻布十番や六本木と違い、歩いている人がサラリーマン多い

# 三軒茶屋
* カフェも多く住みやすそう。
* フリーランスやエンジニアが多そう
* 渋谷までのアクセスが良い
* 色々コミュニティがありそう(三茶.dev)

# 池尻大橋
* 三軒茶屋と同じことを期待していること、こちらの方が落ち着いてて住みやすいとのこと。
* 渋谷までも一駅

# 学芸大学
* 前住んでた。
* 住みやすい。中目黒・渋谷へのアクセスがいい。
* 良いコワーキングスペースを見つけてそこで作業していた。あの環境は良かった。
* 人も多すぎずだけど商店街で活気があっていい街だった。
* 駅徒歩が近ければ今も住んでいそう。また住みたいなとも考えてる

# 恵比寿
- ここで生活の全てが解決する。
- 利便性がとにかく良い。
- 住もうと思う駅では一番良いと思っている。(これ以上のターミナル駅は人が多すぎて住んでも駅を利用するのに苦労しそう)
- というか本当はここに住みたかった。物件が無さすぎて麻布十番になった。人気なんだね。
